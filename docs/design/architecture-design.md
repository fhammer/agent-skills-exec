## 1. 架构概览

### 1.1 设计原则

| 原则 | 说明 |
|------|------|
| **单一职责** | 每个组件职责明确，Coordinator 只负责编排 |
| **开闭原则** | 对扩展开放，对修改封闭 |
| **依赖倒置** | 高层模块不依赖低层模块，都依赖抽象 |
| **渐进复杂** | 从简单场景开始，按需引入分布式能力 |
| **可观测性** | 全链路审计，支持追溯和调试 |

### 1.4 核心设计原则

| 原则 | 说明 | 实现方式 |
|------|------|----------|
| **单一职责** | 每个组件职责明确 | Coordinator只负责编排，Skill只负责业务逻辑 |
| **开闭原则** | 对扩展开放，对修改封闭 | 通过Plugin、Hook机制扩展功能 |
| **依赖倒置** | 高层模块不依赖低层模块 | 通过抽象接口（Agent、Skill、Tool）进行依赖 |
| **渐进复杂** | 从简单场景开始 | 支持单机到分布式渐进部署 |
| **可观测性** | 全链路审计 | 三层上下文审计日志、Metrics监控 |

## 2. 核心抽象层设计

### 2.1 Agent 抽象

```python
from abc import ABC, abstractmethod
from typing import Dict, Any, Optional, AsyncIterator
from dataclasses import dataclass
from enum import Enum


class AgentState(Enum):
    """Agent 状态"""
    IDLE = "idle"
    PLANNING = "planning"
    EXECUTING = "executing"
    SYNTHESIZING = "synthesizing"
    ERROR = "error"


@dataclass
class AgentResponse:
    """Agent 响应"""
    text: str
    structured: Optional[Dict[str, Any]] = None
    metadata: Optional[Dict[str, Any]] = None
    execution_trace: Optional[Dict[str, Any]] = None


class Agent(ABC):
    """
    Agent 抽象基类

    定义 Agent 的核心能力：
    - 处理用户输入
    - 执行规划
    - 协调 Skill 执行
    - 生成响应
    """

    def __init__(self, config: Dict[str, Any]):
        self.config = config
        self.state = AgentState.IDLE
        self._plugins: list = []
        self._hooks: Dict[str, list] = {}

    @abstractmethod
    def process(self, user_input: str, context: Optional[Dict] = None) -> AgentResponse:
        """
        处理用户输入

        Args:
            user_input: 用户输入
            context: 可选上下文

        Returns:
            Agent 响应
        """
        pass

    @abstractmethod
    def process_stream(self, user_input: str, context: Optional[Dict] = None) -> AsyncIterator[str]:
        """
        流式处理用户输入

        Args:
            user_input: 用户输入
            context: 可选上下文

        Yields:
            响应片段
        """
        pass

    @abstractmethod
    def get_state(self) -> AgentState:
        """获取当前状态"""
        pass

    @abstractmethod
    def reset(self) -> None:
        """重置 Agent 状态"""
        pass

    # === 扩展机制 ===

    def register_plugin(self, plugin: "AgentPlugin") -> None:
        """注册插件"""
        self._plugins.append(plugin)
        plugin.on_register(self)

    def unregister_plugin(self, plugin: "AgentPlugin") -> None:
        """注销插件"""
        if plugin in self._plugins:
            self._plugins.remove(plugin)
            plugin.on_unregister(self)

    def add_hook(self, event: str, handler: callable) -> None:
        """添加钩子"""
        if event not in self._hooks:
            self._hooks[event] = []
        self._hooks[event].append(handler)

    def remove_hook(self, event: str, handler: callable) -> None:
        """移除钩子"""
        if event in self._hooks and handler in self._hooks[event]:
            self._hooks[event].remove(handler)

    def trigger_hooks(self, event: str, **kwargs) -> Dict[str, Any]:
        """触发钩子"""
        results = {}
        if event in self._hooks:
            for handler in self._hooks[event]:
                try:
                    result = handler(**kwargs)
                    results[handler.__name__] = result
                except Exception as e:
                    results[handler.__name__] = {"error": str(e)}
        return results


class AgentPlugin(ABC):
    """
    Agent 插件基类

    允许外部扩展 Agent 能力而不修改核心代码
    """

    @property
    @abstractmethod
    def name(self) -> str:
        """插件名称"""
        pass

    @property
    @abstractmethod
    def version(self) -> str:
        """插件版本"""
        pass

    def on_register(self, agent: Agent) -> None:
        """注册时调用"""
        pass

    def on_unregister(self, agent: Agent) -> None:
        """注销时调用"""
        pass

    def on_before_process(self, user_input: str, context: Dict) -> Dict:
        """处理前调用"""
        return {"user_input": user_input, "context": context}

    def on_after_process(self, response: AgentResponse) -> AgentResponse:
        """处理后调用"""
        return response
